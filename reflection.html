Reflection

--> Explain the concept of Responsive Design. What does the term mean?
Responsive design is a method of developing web pages that makes use of flexible layouts, flexible graphics, and media queries in the form of cascading style sheets. The goal of responsive design is to generate web pages that recognise the visitor's screen size and orientation screen and adjust the layout as needed. Responsive design is a somewhat retro approach to web site design that addresses many of the design issues that have arisen as a result of the growth of new sorts of mobile devices. Instead of fixed-width settings, responsive design pages employ X and Y coordinates on a grid for layout and mathematical percentages for pictures. Using percentages instead of fixed-width parameters with a grid layout results in a more fluid layout that adapts to the size of the display.

--> How does responsive design relate to web technologies?
Responsive web design, often known as RWD design, is a current web design strategy that allows websites and pages to render (or display) across all devices and screen sizes by changing to the screen automatically, whether it's a desktop, laptop, tablet, or smartphone. CSS (Cascading Style Sheets) are used in responsive web design to serve different style properties based on the user's device's screen size, orientation, resolution, colour capability, and other features. CSS properties that are relevant to responsive web design include viewport and media queries.
